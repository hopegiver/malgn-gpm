## 시스템 개요

### 핵심 철학
전략 → 목표 → 실행 → 점검 → 평가가 **단절 없이** 하나의 흐름으로 연결되는 목표 중심 운영 시스템

### 목표 트리 (Goal Tree)
```
미션/비전
 └─ 전사 전략목표 (BSC)
     └─ 팀 KPI
         └─ 개인 MBO
             └─ 주간계획 / 일일업무
                 └─ 평가 (중간/최종)
```

### 시스템 특징
- ✅ 목표-업무-평가 완전 연결
- ✅ 실시간 데이터 기반 평가
- ✅ 전사 목표 가시성 확보
- ❌ 보고를 위한 보고 제거
- ❌ 연말 기억 평가 제거
- ❌ KPI/MBO 분리 운영 제거

## 메뉴 구성 철학

### 기본 원칙
1. **기능별이 아닌 시간·목표 흐름별 구성**
2. **목표 트리를 시스템의 뿌리로 설정**
3. **모든 업무는 상위 목표와 연결 필수**
4. **평가는 데이터 자동 집계 중심**

### 메뉴 설계 방향
- HR 관점(평가, 보고, 일지) → 목표 중심 관점으로 전환
- 독립적 기능 → 연결된 프로세스
- 사후 평가 → 실시간 모니터링

## 전체 메뉴 구조 (2 Depth)

### 역할별 메뉴 구성 철학
- **직원**: 쓰기 쉽고, 보고하기 편하게
- **관리자**: 한눈에 보고, 빠르게 개입하기
- **개인 성장**: 단순 평가가 아닌 역량/스킬 성장 중심

### 🎨 메뉴 UX/UI 가이드

#### 1. 빠른 접근 (Quick Access)
모든 역할 공통으로 화면 상단에 배치:
```
[🔔 알림]  [⚡ 빠른 작성]  [📊 내 현황]  [🚨 위험 신호(관리자만)]
```

#### 2. 자주 쓰는 기능 바로가기
**직원용**:
- 오늘의 업무 작성 (1-click)
- 이번 주 보고 작성 (1-click)
- 나의 목표 달성률 (1-click)

**관리자용**:
- 위험 신호 확인 (1-click)
- 팀 현황 보기 (1-click)
- 미작성자 독촉 (1-click)

#### 3. 알림 시스템
```
🔴 긴급 (즉시 개입 필요)
- 목표 달성률 30% 미만 (분기 기준)
- 3주 이상 주간보고 미작성
- 위험 이슈 등록됨

🟡 주의 (확인 필요)
- 1:1 미팅 2개월 이상 없음
- 목표 대비 실행 기록 부족
- 평가 기한 1주일 전

🟢 정보
- 새 피드백 도착
- 목표 달성률 상승
- 팀 목표 업데이트
```

#### 4. 색상 시그널 (신호등 시스템)
```
🔴 빨강 (위험): 달성률 40% 미만 또는 지연
🟡 노랑 (주의): 달성률 40~70%
🟢 초록 (양호): 달성률 70% 이상
⚪ 회색 (미시작): 아직 시작 안 함
```

### 2️⃣ 목표 (Goals)

**목적**: 시스템의 핵심, 전사~개인 목표 연결

#### 2-1. 전사 KPI

**주요 기능**
- BSC 기반 KPI 목록 관리
- 연/반기 목표값 설정
- 담당 임원/부서 지정
- 달성률 자동 집계
- 추이 그래프

**데이터 항목**
```
- KPI 명
- BSC 관점
- 측정 지표
- 목표값 (연간/반기)
- 담당 임원
- 담당 부서
- 가중치
- 현재 달성률
```

**권한**: 임원, 전략기획팀

---

#### 2-2. 팀 KPI

**주요 기능**
- 전사 KPI와 연결 (필수)
- 팀별 핵심 지표 관리
- 팀 KPI ↔ 개인 MBO 매핑
- 팀 성과 대시보드

**데이터 항목**
```
- 팀 KPI 명
- 연결된 전사 KPI
- 측정 지표
- 목표값
- 팀장
- 팀원
- 가중치
- 달성률
```

**특징**
- 전사 KPI 없이 팀 KPI 생성 불가
- 하위 개인 MBO 자동 연결

**권한**: 팀장, 부서장

---

#### 2-3. 개인 MBO

**주요 기능**
- 개인 목표 설정

## 기대 효과

### 조직 관점
✅ **전략 정렬 (Strategic Alignment)**
- 전사 전략이 개인 업무까지 연결
- "왜 이 일을 하는가?"에 대한 명확한 답

✅ **실시간 모니터링**
- 연말 아닌 매일 성과 확인
- 조기 리스크 감지 및 대응
- 🚨 위험 신호 시스템으로 선제적 개입

✅ **데이터 기반 의사결정**
- 감이 아닌 데이터로 판단
- 목표-실행-성과 인과관계 분석
- 전사 역량 포트폴리오 기반 육성 전략

### 관리자 관점
✅ **투명한 관리**
- 팀원 목표/진척 실시간 확인
- 피드백 근거 확보
- 1-click으로 즉시 개입 가능

✅ **공정한 평가**
- 기억이 아닌 기록 기반
- 평가 이의 최소화
- 자동 집계로 평가 시간 80% 단축

✅ **효과적 코칭**
- 중간 점검 통해 방향 수정
- 연말 깜짝 평가 방지
- 역량 데이터 기반 맞춤형 육성

### 직원 관점
✅ **명확한 방향성**
- 내 업무가 회사 목표에 어떻게 기여하는지 명확
- 우선순위 판단 용이
- 목표 트리로 전체 맥락 이해

✅ **공정한 평가**
- 내가 한 일이 그대로 평가에 반영
- 투명한 평가 프로세스
- 실시간 달성률로 예측 가능

✅ **성장 중심 문화** ⭐ 신규
- 단순 평가가 아닌 역량 개발 초점
- 피드백 → 역량 점수 자동 반영
- 나의 성장 여정 시각화
- 학습 계획과 MBO 연결
- 멘토링 시스템 체계화

### 시스템 관점
✅ **단절 제거**
- 전략-목표-실행-평가 완전 연결
- 정보 손실 최소화
- 목표-업무-역량 삼각 연결

✅ **자동화**
- 수작업 집계 제거 (80% 감소)
- 실시간 계산 및 분석
- 위험 신호 자동 감지

✅ **확장성**
- 조직 성장에 따른 유연한 확장
- 다양한 평가 방식 적용 가능
- 역량 프레임워크 커스터마이징

### 사용성 관점 ⭐ 핵심
✅ **쓰기 편함 (직원)**
- 일일 업무: 30초 입력
- 주간 보고: 10분 작성 (자동 요약)
- 모바일 최적화

✅ **보기 편함 (관리자)**
- 한 화면에 모든 위험 신호
- 신호등 색상으로 즉시 파악
- 1-click 액션

✅ **성장 동기 (개인)**
- 역량 레이더 차트로 성장 시각화
- 피드백이 즉시 점수에 반영
- 목표와 학습이 연결됨

## 성공 핵심 요소 (Critical Success Factors)

### 1️⃣ 간편한 입력 (직원)
- ✅ 일일 업무: 3필드만 입력 (30초)
- ✅ MBO 드롭다운 선택으로 자동 연결
- ✅ 모바일에서도 쉽게 작성
- ❌ 복잡한 양식, 긴 텍스트 입력 제거

### 2️⃣ 즉시 파악 (관리자)
- ✅ 위험 신호 대시보드 최상단 배치
- ✅ 신호등 색상 (🔴🟡🟢)으로 즉시 이해
- ✅ 1-click 액션 (독촉, 1:1 예약, 개입)
- ❌ 복잡한 리포트, 여러 화면 클릭 제거

### 3️⃣ 성장 동기 (개인)
- ✅ 역량 레이더 차트 실시간 업데이트
- ✅ 피드백 → 점수 자동 반영 (즉각 보상)
- ✅ 스킬 트리로 게임화 요소
- ✅ 평가가 아닌 성장에 초점

### 4️⃣ 자동화 (시스템)
- ✅ 일일 로그 → 주간 요약 자동 생성
- ✅ 목표 달성률 실시간 계산
- ✅ 위험 신호 자동 감지 & 알림
- ✅ 역량 점수 자동 집계

## 결론

> 이 시스템은 단순한 "성과관리 시스템"이 아니라,
> **"조직의 목표가 실제 행동으로 내려오고,
> 행동이 다시 성과와 성장으로 증명되는 통합 운영 시스템"**입니다.

### 핵심 가치

**목표 중심**
- 전사 전략부터 개인 업무까지 완전 연결
- "왜 이 일을 하는가?"에 대한 명확한 답

**데이터 기반**
- 기억이 아닌 기록으로 평가
- 실시간 모니터링과 선제적 개입

**성장 중심** ⭐ 핵심
- 단순 평가를 넘어 역량 개발에 초점
- 피드백-학습-실행이 선순환 구조

**사용자 중심**
- 직원: 쓰기 쉽고 성장 동기 부여
- 관리자: 보기 쉽고 빠른 개입 가능
- 임원: 전사 현황 한눈에 파악

### 차별화 포인트

| 기존 시스템 | 이 시스템 |
|-----------|----------|
| 연말에 평가만 | 매일 성장 기록 |
| 보고를 위한 보고 | 목표 연결 필수 |
| 기억 기반 평가 | 데이터 자동 집계 |
| HR 도구 | 운영 시스템 |
| 평가 중심 | **성장 중심** ⭐ |
| 복잡한 입력 | 30초 간편 입력 |
| 사후 대응 | 실시간 위험 감지 |

### 한 문장 정의

**"성과운영 시스템은 전략-목표-실행-성장-평가를 하나로 연결하여, 조직과 개인이 함께 성장하는 목표 중심 운영 플랫폼이다."**
